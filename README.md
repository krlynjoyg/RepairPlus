# RepairPlus: Smart Repair Assistance App

## Overview
RepairPlus helps users find repair tutorials and nearby repair shops using Flutter and Firebase.

## Tech Stack

- Flutter (Frontend)
- Firebase Firestore (Database)
- Firebase Authentication
- Google Maps API, iFixit API, YouTube Data API

## Features
- **Repair Tutorials** – Fetches video and reading guides from YouTube and iFixit APIs.  
- **Nearby Repair Shops** – Uses Google Maps API and OpenStreetMap Overpass API to display local repair centers.  
- **Saved Tutorials** – Allows users to bookmark/star tutorials for later.  
- **History Tracking** – Tracks viewed and saved tutorials in real-time via Firebase Firestore.  
- **User Authentication** – Secure login and registration with Firebase Auth.  

## Note
The current version of RepairPlus is compatible **only with Android devices**.  
> iOS support is not yet implemented.
